# EngineerPocket Rhino8 plugin
A simple and efficient toolbar plugin and gha plugin for structural engineers working with Rhino8, designed to help quickly import the skeleton and 3D geometry of models from SAP2000, ETABS, and other software into RHINO. It also provides real-time conversion between Sap2000 and ETABS models within activated program. Other funny little features, e.g. music player, video player, locking commandsinput to English, restarting Rhino, are continuously being added. Plugins are developed in C#. 

SetDotNetRuntime in Rhino8 need to be 'netframework' temporarily. 

Net7, Net8 will be supported in subsequent versions.
